### 配置

- repo：用户名/仓库名;
- branch：默认为 master 即可；
- token：刚才生成的令牌；
- path：可自定义，但是要填，否则可能导致上传失败；
- customPath：图片归档方式，以年、年月……
- customUrl：建议保持默认；粗体
